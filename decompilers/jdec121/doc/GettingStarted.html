Description :

 Lets gets started. It just takes 2 mins to use jdec for  most common task - Decompile class file.

For all  "routes" to decompile a file ....
    
     The user has to configure jdec. Please  select the menu item "JDec(Decompiler) Config" to do this. 
     Just update all "***REPLACE_ME***"  with valid values. Ofcourse you dont have to actually  have to
     give values for java_class_file and jar_file_path.         

    [Note:] For more details refer to configuration link.

Now you are ready to use jdec.

To decompile a class file follow any of the below approaches:

  1. Click File->New Decompiler Task -> Decompile Class File
  2. Select a class file from the explorer tree(Dir tab) shown in left tab and click.
  3. Enter a value in Config window for java_class_file and click Run buttion [2nd button from left] 
        [Note: Ensure the JDec_Option is set for decompiling]
    4. Click the favorite tab to select a class file.
        [Note: Add Favorites by clicking Edit Favorite Link from utilities menu option]
    5. Click File->Class In Archive Task->Follow steps
