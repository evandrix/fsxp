Description :

For jdec 1.2 release there are 4 tasks . 2 new tasks have been added for
this release. Also the jar task has been updated. It also has a new name
- archive task. This is take into account other type of formats also.

4 tasks are:
  • Decompile class task
  • Decompile archive task
  • Class in arhive task
  • Diff 2 versions task

Again I would like to repeat. Please ensure the config.property values are correct before starting any task.

1.
 Decompile Class: Allows you to choose a class file from the file system.

2.
 Decompile Archive Task: Allows you to choose an archive file. By default an archive file is zip or jar file.
 Please alter the settings(below preferences from edit menu) to define new types of arhive files.
 I would request the user to go through the filter documentation before attempting to decompile an archive file.
 Updating the filters can result in faster decompilation and also can avoid decompiling unnecessary classes.

3.
 Class in archive task. Allows the user to mention a class file and a jar containing that class file.
 jdec will automatically read the bytecode for that file and decompile that class.

4.
 Diff 2 versions task. Allows the user to specify 2 class files(same class but in different locations) either from
 a folder or from a jar. This will be slightly experimantal for this release i.e. the intention is to provide
 full UI support for comparing 2 files ; just like in a normal diff tool. This may not have all features
 when jdec 1.2 is released.
