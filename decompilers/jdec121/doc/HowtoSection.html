Description:

This section explains some "how do i do with jdec" actions...

    Category : Appearance

     1.   How do I change the Syntax Coloring and/or font?


  •  Click Config-->Syntax and Coloring option to update
  •  Follow the steps. They are intuitive enough for  user yo update his settings.


      2.   How do  I disable Syntax Highlighting ?


  •  Click Edit--->Preferences
  •  Choose No in dropdown options for "Enable Syntax  Highlighting"
  •  Restart Jdec


    3.  How  do I change the look and feel ?


  •  Click Edit--->Preferences
  •  Choose an option in dropdown options for "Default Look n feel"
    

[NOTE: Click the button for LNF HINT for more information]

    4.  How do I launch jdec with custom look and feel ?


  •  Click Edit--->Preferences
  •  Click the button for LNF HINT for more information.


    5.  The editor/output window appears  to be small. How do I resize it?


  • Click Window--->Hide Console Window. OR
  • Drag the Console window down.


    6.  How do I get the default tabs[output] to be shown[Too many tabs are open in UI] ?


  •  Click window--->Reset Tabs


    7.  How do I close an output tab?


  •  It is not possible to close any tab explicitly. However you can see the default starting tabs again and close others
  •  Click window--->Reset Tabs



   Category : Configuration

     1.How do I  change jdec configuration details?


         ...Refer Configuration Help page

     2.Will jdec work if configuration details are not updated.?

  •  One word answer -> No.
  •  Initially this configuration details were only applicable for Run Image button.
         However for some reason during development of UI, all "routes" to decompile
         were linked with configuration.
  •  A future release may restrict configuration details applicable only for Run Image Button



   Category : General
   
     1. How do I look up for any errors ?


  • Depending on how the log mode has been configured, the log will appear.
  • For the UI route file output is the only option.Open that file
  • Set the debug level to 2 for more details
  • For the command line version console mode can also be set.
    


    2. How do I report any error?


  • Please visit jdec home page and click on "Report a problem" link. OR
  • Please visit sourceforge site for Jdec and submit a bug
  • Email the author directly for any help or to inform him of error.


    3. How do I see the output in my favorite Editor?


  • Click Utilties-->Show in custom editor


    4. How do I configure my favorite editor?

  • Click Edit--->Preferences option.
  • Update the value for "Set your favorite Editor Path"


    5. How do I compile a java file ?


  • Click Utilities--->Compile
  • Support is not very good for this option


    6. How do I run a java file ?


  • Click Utilities--->Run
  • Support is not very good for this option


    7. How do I add Favorite Folders?

  • Click Utilities --> Edit Favorite List.
  • Restart Jdec
  • Click on Favorite Tabs


     Category: Decompiler Tasks

     1. I  Have decompiled a class file and compiled the source separately. what is the quickest way to repeat the task ?


  •  Click on File --> Refresh.


    2 . How do I selectively decompile contents of a jar file? 

  • Please refer to Filter documentation.
  


     3. How do I selectively decompile methods of a class file

  • Please refer to Filter documentation.


     4. How do I navigate to any method in a decompiled output.


  • Decompile a class file
  • Select a method in class methods tab OR
  • press ctrl-m to see in pop-up window.
  • In the pop-up window select any method and either click  on Actions OR clipboard
  
    5. My jar is very big. How do I make the best use of jdec?
     
      Follow any of these approaches
  • First Configure the jar filter level options(Refer filter docs)
  • Second Configure archive level tasks(Refer filter docs)
  • Decompile the entire  jar by clicking yes when jdec shows a prompt for jar option
     
              OR
                  
  • First Configure the jar filter level options(Refer filter docs)
  • Second Configure archive level tasks(Refer filter docs)
  • Decompile class files selectively by clicking no when jdec shows a prompt for jar option.
         
             OR
    • If you know which class files exactly you want to decompile then...
    • Select the task "Class in Archive task" and follow the steps.
    • Repeat above step for all desired classes.
